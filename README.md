# Portfolio Website - CodeAlpha Task 3

## Project Description
A responsive personal portfolio website built with HTML, CSS, and JavaScript as part of the CodeAlpha internship program. This portfolio showcases skills, projects, and contact information with a modern, clean design.

## Live Demo
[View Live Portfolio](https://yourusername.github.io/CodeAlpha_Portfolio/)

## Features
- ✅ **Responsive Design**: Works on all devices (mobile, tablet, desktop)
- ✅ **Modern UI/UX**: Clean, professional design with smooth animations
- ✅ **Interactive Elements**:
  - Smooth scrolling navigation
  - Animated skill bars
  - Project cards with hover effects
  - Contact form with validation
- ✅ **Portfolio Sections**:
  - Hero/Home section with profile
  - About section with skills display
  - Projects showcase with filterable tags
  - Contact form and information
- ✅ **Performance Optimized**: Fast loading with optimized images

## Technologies Used
- **HTML5**: Semantic markup
- **CSS3**: Flexbox, Grid, CSS Variables, Animations
- **JavaScript (ES6)**: Interactive functionality
- **Font Awesome**: Icons
- **Google Fonts**: Typography

## How to Use
1. Download or clone this repository
2. Open `index.html` in any modern web browser
3. The website is fully functional with no additional setup required

## Customization
Replace the following placeholders with your information:
- `[Your Name]` - Your name throughout the website
- `[Your City, Country]` - Your location
- Profile image URL
- Contact information
- Project details and images
- Social media links

## Browser Support
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 16+

## Deployment
This project is deployed using GitHub Pages. To deploy your own:
1. Push code to GitHub repository
2. Go to Repository Settings → Pages
3. Select "Deploy from branch" → main branch
4. Your site will be live at: `https://username.github.io/CodeAlpha_Portfolio/`
